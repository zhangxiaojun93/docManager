    消息已读状态更改接口文档

    消息平台文档管理系统 2019-05-26 消息更新 接口

    # 1. 接口描述

    • 接口名称 消息已读状态更改接口
    • 接口说明 该接口可将制定消息更改为已读状态
    • 接口地址

    调用示例

    http://localhost:8081/git/api/message/updateMessageRead
    （消息平台地址 + api/message/updateMessageRead）

    • 接口编码
      UTF-8
    • 支持协议 HTTP(POST)

    # 2. 请求参数

    请求参数 参数类型 参数说明 是否必选
    messageItemIds String 要设为已读的消息id，多个id用“,”隔开 是

    # 3. 返回参数

    返回参数 参数类型 参数说明
    requestId String 请求ID全局唯一
    code String 状态码
    msg String 状态信息
    success boolean 是否执行成功

    # 4. 调用示例

        fnUpdateMessageRead(record){
        let vue = this;
        //解决axios请求传入参数是json问题
        let data= new URLSearchParams();
        if(record == null){
            vue.$message.error("获取当前数据出错！");
            return;
        }else{
            data.append('messageItemIds',record.id);
        }
        Axios({
            method: 'post',
            url: faceConfig.basePath + "/api/message/updateMessageRead",
            withCredentials:true,//携带cookies
            data: data
        }).then((response) => {
            let resultInfo = response.data;
            if(resultInfo.code === "200"){
                vue.loadData();
            }else{
                vue.$message.error(resultInfo.msg);
            }
        });
    }

    注：调用该接口必须传入指定消息的id，所以该接口目前暂不支持对websocket接收的消息进行处理

    # 5. 返回示例

    • 示例：
          {
    	"msg": "更新成功！",
    	"code": "200",
    	"requestId": "b2e5966f-af4f-47ef-88f2-a7ef1ad09252",
    	"success": true
    }

# 支付宝消息发送说明

# 1. 功能概述

  • 功能概述 支付宝生活号消息推送功能整合了支付宝生活号消息推送api，为接入系统提供生活号消息推送功能。可推送消息类型包括异步单发消息（客服模式）、单发模板消息、群发消息和分组消息
    • 异步单发 对用户消息的回复功能，也可以使用该接口实现客服等扩展功能。 注：用户主动与生活号交互的48小时之内，可以使用该消息类型向该用户进行消息推送。
    • 单发模板消息 模板消息是指如银行动账提醒、航空公司值机提醒等对消费者非常有价值的消息类型，模板消息符合特定的格式，无需用户发生交互就可以主动向用户推送。
    • 群发消息 向所有关注该生活号的用户发送消息
    • 分组消息 向指定分组中的关注用户发送消息

# 2. 接入流程

  1. 新增生活号配置
    • 渠道管理-支付宝生活号配置-新增-填写支付宝生活号应用信息
  2. 新增接入系统授
    • 接入管理-接入系统管理-选择自己的系统-配置系统渠道 注：若系统渠道权限中不包含支付宝，则不能发送支付宝消息
  3. 系统绑定发送消息生活号配置
    • 接入管理-系统绑定-新增-选择第一步添加的生活号 注：若未配置生活号，则会查询生活号配置中的默认配置，若无则不能发送消息
  4. 按照消息发送api提供消息主体和消息子体中所需内容，再调用接口即可

# 3. 接口说明

  • 生活号消息统一调用消息发送接口，参数特殊说明如下
    • 发送类型（sendType）固定为“5”，支付宝消息
    • 消息标题(messagetTitle)和消息内容(messagetContent)必传 注：消息内容可能为Json数组，详见各消息说明
    • 消息类型（messageType）分为文本消息、图文消息、模板消息 注：当需要消息子体的参数时，如果消息子体中的messageType为空，会以消息主体中messageType为准
    • 模板编号（templateCode）此模板编号针对本消息平台模板编号，与支付宝模板消息无关
    • 是否发送所有人（isToAll）当该值为true，即为群发消息
    • 是否分组发送（isGroupSend）当isToAll为false时该值有效，为true时，即为分组消息
    • 批量发送内容是否一致（isSameBatchContent）当为群发或者分组发送时默认为true，当为单发模板消息或异步单发时，会根据该字段为true或者false从消息主体或消息子体获取消息标题和内容
    • 文件表单（file）所有文件都存在该字段中 注：后台根据文件名称关联文件内容，所以文件名称不可重复
    • 消息子项（items）只在异步单发消息、单发模板消息中才会有值

# 4. 调用说明

# 1. 异步单发消息

  • 说明 当isToAll为false、isGroupSend为false且messageType为为文本消息或图文消息时即为异步单发消息,当isSameBatchContent为true时，以消息主体中messageType为为准，当isSameBatchContent为false时以item中的messageType为准，当其为空时以消息主体中的messageType为准
  • 消息主体参数
请求参数 参数名称 参数类型 详细说明
sendType 发送类型 String "5"
isToAll 是否发送所有人 Boolean false
isGroupSend 是否分组发送 Boolean false
isSameBatchContent 批量发送内容是否一致 Boolean 当为true，取消息主体中内容，为false取消息子体中内容

  • 消息子体参数
请求参数 参数名称 参数类型 详细说明
toUserId 接收者支付宝openId String 一般当用户触发某个既定事件时才能获得该用户的openId
messageContent 消息内容 String、Json数组 文本消息时为消息内容，图文消息时为Json数组，参考图文消息
messageType 消息类型 String 文本消息、图文消息

# 3. 群发消息

  • 说明 当isToAll为true时即为群发消息
  • 参数
请求参数 参数名称 参数类型 详细说明
messageTitle 消息标题 String 必传
messageContent 消息内容 String、Json数组 文本消息时为消息内容，图文消息时为Json数组，参考图文消息
sendType 发送类型 String "5"
messageType 消息类型 String 文本消息、图文消息
isToAll 是否发送所有人 Boolean true
isGroupSend 是否分组发送 Boolean 任意值
isSameBatchContent 批量发送内容是否一致 Boolean true

  • 示例
      {

    "isSameChannelContent": false,

    "ticket": "C6C1A6106FF32E5ADF6F99A5F7734C02",

# 4. 分组消息

  • 说明 当isToAll为false且isGroupSend为true时即为分组消息，此时需传入tagId
  • 参数
请求参数 参数名称 参数类型 详细说明
messageTitle 消息标题 String 必传
messageContent 消息内容 String、Json数组 文本消息时为消息内容，图文消息时为Json数组，参考图文消息
sendType 发送类型 String "5"
messageType 消息类型 String 文本消息、图文消息
tagId 分组编码 String 支付宝关注用户分组的编码
isToAll 是否发送所有人 Boolean false
isGroupSend 是否分组发送 Boolean true
isSameBatchContent 批量发送内容是否一致 Boolean true

  • 示例
      {
    "isSameBatchContent": false,//指定false无效
    "businessCode": "si_yzm",
    "isSameChannelContent": false,
    "sendName": "王小二",
    "ticket": "C6C1A6106FF32E5ADF6F99A5F7734C02",
    "ssystemCode": "TEST1",
    "sendUserid": "2",
    "isToAll": false,
    "areaCode": "512000",

    "messageTitle": "群发消息标题",

# 4. 特殊说明

  • 文本消息

    • 示例
             messageContent = "文本消息的内容";

  • 图文消息 注：当为图文消息时，参数是Json数组

    • 参数
    请求参数 参数名称 参数类型 是否必填 详细说明
    title 消息标题 String 是 图文消息标题
    desc 消息正文 String 是 图文消息内容
    image_url 图片链接 String 特殊可选 对于多条图文消息的第一条消息，执行发送消息时必须有该字段。该字段可以直接传入，也可以上传附件，通过附件自动生成
    url 链接地址 String 是 点击消息跳转的链接
    action_name 链接文字 String 否
    isNeedUpLoadFile 是否需要文件上传 Boolean 是 是否需要文件上传标识，true为是，false为否
    fileName 文件名称 String 特殊可选 当需要文件上传时必填，提供文件名称，方便在上传文件列表中查询对应文件

    • 示例
            messageContent = [{
         	" title": "图文消息标题",
         	"desc": "图文消息内容",
         	"image_url": "http: //example.com/abc.jpg",
         	"url": "https://www.example.com/b.php",
         	"action_name": "查看详情",
         	"isNeedUpLoadFile": "1",
         	"fileName": "123"
         }]

  • 模板消息

    • 参数
    请求参数 参数名称 参数类型 是否必填 详细说明
    template_id 模板编号 String 是
    context 上下文 Json 是 消息模板上下文，即模板中定义的参数及参数值
    • context
    请求参数 参数名称 参数类型 是否必填 详细说明
    head_color 头部颜色 String 是 顶部色条的色值
    url 链接地址 String 是 点击消息后承接页的地址
    action_name 链接文字 String 是
    keyword1 关键字 Json 可选 模板中占位符的值及文字颜色
    keyword2 关键字 Json 可选 模板中占位符的值及文字颜色
    first 关键字 Json 可选 模板中占位符的值及文字颜色
    remark 关键字 Json 可选 模板中占位符的值及文字颜色
    • keyword1、keyword2、first、remark
